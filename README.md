# 個人房貸試算器
輸入欲購屋的總金額（單位：萬元）、預計準備的自備款百分比、銀行的年利率百分比、預計貸款的長度（單位：年)、若有需求，請輸入前幾年僅繳利息的年數；若無預設值為0。

確認資訊無誤後，點擊右側的「開始計算」按鈕，下方區域將立即顯示試算結果。

若需進行另一組試算，點擊「清除重新輸入」按鈕，所有欄位將回復至預設狀態。
